# Support

CC Theme is community-maintained pre-release software and is provided without a
service-level guarantee.

Before opening an Issue:

1. Read the root README and the affected subproject README.
2. For `mac-codex`, run the doctor/status commands and review
   [`adapters/mac-codex/docs/USAGE.zh-CN.md`](adapters/mac-codex/docs/USAGE.zh-CN.md).
3. Search existing Issues for the same client version and symptom.
4. Remove account names, task content, private paths, tokens, logs, and personal
   media from anything you share.

Use the Bug Report form for reproducible defects, the Compatibility Report form
after a desktop-client update, and the Feature Request form for proposals.
Questions about modifying unsupported third-party binaries or bypassing platform
security controls are outside the project's support scope.

Security vulnerabilities must follow [`SECURITY.md`](SECURITY.md) and must not be
reported publicly.
