# Repository baseline

This repository starts a new history at product version `0.1.0`.

The repository intentionally excludes:

- every previous `.git` directory, branch and release artifact;
- installed runtime copies, Application Support state, logs and backups;
- historical screenshots and one-time absolute paths;
- user-provided or named-person images and videos;
- sponsor or promotional copy;
- Windows code and menu-bar/tray switching.

Production themes and media are owned by the independent CC Theme resource layer.
This Adapter repository and its release contain no bundled/default/fallback theme;
an empty installation remains in the native Claude appearance until a verified
external Theme Package is selected.
